Adaptive Platonic Catalogue

This research repository extends the classical Platonic solid catalogue under adaptive π geometry. It encodes the existence rule with the ratio ρ = πv/πf and tests which regular {n,q} pairs satisfy 1/n + (ρ/q) > 1/2. When ρ = 1 the classical condition recovers exactly five Platonic solids; at other ratios new families appear. The project includes scripts to generate tables of admissible polyhedra, render patches, produce phase diagrams and create equation cards.
