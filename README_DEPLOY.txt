# AdaptiveCAD landing site (GitHub Pages)

1) Copy this `site/` folder and `.github/workflows/pages.yml` to your repo root.
2) Commit to `main`. GitHub Actions will deploy to Pages automatically.
3) Replace:
   - `site/assets/hero.mp4` with your final Kickstarter hero video
   - `site/assets/hero_poster.jpg` as poster frame
   - `YOUR_KICKSTARTER_URL` in `site/index.html` (3 places)
